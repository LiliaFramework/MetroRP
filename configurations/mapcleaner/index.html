MapCleaner

Configuration for Map Cleaner Module.

Tables

Configuration

This table defines the default settings for the Map Cleaner Module.

Fields

  • MapCleanerEnabled

    Indicates whether the map cleaner is enabled | bool

  • ItemCleanupTime

    The time in seconds before items are cleaned up | number

  • MapCleanupTime

    The time in seconds before the entire map is cleaned up | number

  • MapCleanerEntitiesToRemove

    A list of entity types that will be removed during map cleanup | table
